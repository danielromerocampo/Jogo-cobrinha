# 🐍 Snake Game Modern

## 🎮 Sobre o Projeto

Este projeto recria o clássico jogo da cobrinha com uma abordagem moderna, oferecendo:

* Interface elegante e responsiva
* Animações suaves
* Controles por teclado
* Controles visuais para dispositivos móveis
* Sistema de pontuação em tempo real
* Tela de Game Over
* Reinicialização rápida da partida
* Código organizado e comentado

O objetivo é simples:

> Coma o máximo de comida possível sem bater nas paredes ou em seu próprio corpo.

---

## ✨ Funcionalidades

### 🐍 Mecânica do Jogo

* Movimento fluido da cobrinha
* Crescimento progressivo ao consumir comida
* Geração aleatória de alimento
* Sistema de pontuação
* Detecção de colisão com paredes
* Detecção de colisão com o próprio corpo
* Reinício instantâneo após derrota

### 🎨 Interface Moderna

* Design inspirado em jogos web modernos
* Layout centralizado
* Painel de pontuação elegante
* Overlay de início de jogo
* Overlay de Game Over
* Efeitos visuais suaves

### 📱 Responsividade

* Funciona em desktops
* Funciona em tablets
* Funciona em smartphones
* Canvas adaptável ao tamanho da tela

### 🎮 Controles

#### Desktop

| Tecla | Ação                |
| ----- | ------------------- |
| ↑     | Mover para cima     |
| ↓     | Mover para baixo    |
| ←     | Mover para esquerda |
| →     | Mover para direita  |

#### Mobile

Botões direcionais interativos:

* ▲ Cima
* ▼ Baixo
* ◀ Esquerda
* ▶ Direita

Com feedback visual ao toque.

## 📂 Estrutura

O projeto foi construído em um único arquivo:

```text

│
├── HTML
├── CSS (embutido)

```

Isso facilita:

* Hospedagem
* Compartilhamento
* Estudos
* Customizações

## 🚀 Como Executar

### Método 1

1. Faça o download do arquivo:

```text
index.html
```

2. Abra o arquivo em qualquer navegador moderno.

Pronto.

### Método 2

Usando VS Code:

```bash
git clone https://github.com/seu-usuario/snake-game.git
```

Abra a pasta e execute:

```bash
index.html
```

ou utilize a extensão:

```text
Live Server
```
